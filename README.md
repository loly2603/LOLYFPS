# LOLYFPS

Este proyecto es un juego de tipo First Person Shooter (FPS) desarrollado en Unity. 
Los personajes y el mundo del juego fueron creados utilizando assets de Unity, integrados 
y organizados dentro de una escena jugable.

## Características
- Juego en primera persona (FPS).
- Personajes creados con assets de Unity.
- Mundo/escena construida con recursos de Unity.
- Escena principal lista para jugar y explorar.

## Cómo ejecutar
1. Clonar este repositorio.
2. Abrir el proyecto en Unity Hub (versión recomendada: 2021.3 LTS o superior).
3. Ejecutar la escena principal (`MainScene.unity`).

## Publicación
El proyecto también está disponible en Itch.io como versión jugable.
